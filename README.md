## **Приложение для проектирования схем реляционных баз данных**

## **Инструкция пользователя**

Приложение для проектирования схем реляционных баз данных предназначено
для управления диаграммами пользователя, облегчая при этом
кооперирование пользователей и внедрение построенных схем в выбранную
СУБД.

При входе в систему пользователю следует зарегистрироваться или
авторизоваться в системе.

Для того, чтобы зарегистрироваться необходимо:

1.  нажать кнопку «Зарегистрироваться» на главной странице;

2.  ввести данные учетной записи;

3.  нажать кнопку «Создать аккаунт».

1.  нажать кнопку «Войти» на главной странице;

Рисунок 2 - Страница авторизации

Если пользователь не помнит пароль, то необходимо:

1.  нажать кнопку «Восстановить пароль» на странице авторизации;

2.  ввести Email;

3.  нажать кнопку «Продолжить»;

4.  ввести код для восстановления пароля;

5.  нажать кнопку «Продолжить»;

6.  ввести новый пароль и подтверждение нового пароля;

7.  нажать кнопку «Восстановить пароль».

Рисунок 7 - Страница импорта диаграммы

После создания или импорта диаграммы система выводит страницу работы с
диаграммой. На данной странице возможно изменять название, код описания
и тему диаграммы. Для сохранения изменений необходимо нажать сочетание
клавиш «Ctrl-S». В редакторе кода возможны дублирование с помощью
сочетания клавиш «Ctrl-D» и комментирование с помощью сочетания клавиш
«Ctrl-/» выделенной части текста.

Также на странице есть возможность взаимодействия с диаграммой. При
помощи колесика мыши или кнопок в правом нижнем углу диаграммы можно
управлять размером изображения диаграммы. Перемещать изображение
диаграммы можно, зажимая левую клавишу мыши и перемещая курсор.
